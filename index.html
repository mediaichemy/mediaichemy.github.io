Getting Started

Follow these steps to set up and start using mediaichemy:

Clone the Repository

git clone https://github.com/your-repo/mediaichemy.git
cd mediaichemy

Install Dependencies

pip install -e .

Set Up API Keys

Configure API keys for supported providers. See the Registering AI Providers section for details.
